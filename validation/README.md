# 四つの平均--分散解概念の検証

付録A.3に対応する検証コードです。検証の意味を混同しないため、外部ベンチマークと内部数値監査を分けています。

| モデル | 検証 | 主な判定 |
| --- | --- | --- |
| PCMV | van Staden, Dang and Forsyth (2021) Table 5.1の外部再現 + 基準ケース内部監査 | 公表32項目、Monte Carlo、後退・前進モーメント、質量保存 |
| DOMV | 保存済み後退方策から独立に前進分布を再構成 | 一次・二次モーメント、質量保存、上端境界 |
| cTCMV | 保存済み均衡方策から独立に前進分布を再構成 | 一次・二次モーメント、質量保存、上端境界 |
| dTCMV | 基準ケース内部監査 + 入れ子上端格子監査 | モーメント、質量、`x_max=900` と `x_max=2100` の主要統計量差 |

DOMV、cTCMV、dTCMVには、PCMV Table 5.1と同じ条件・同じ統計表を公表した外部資料がないため、外部再現を装っていません。代わりに、後退計算とは別の前進分布伝播、確率質量、境界超過量、入れ子格子安定性を機械判定します。

## 実行

リポジトリのルートで実行します。

```bash
python validation/pcmv/external_validation_vanstaden2021_pcmv.py
python validation/run_all_validations.py
```

主な出力は `results/validation/` に保存されます。いずれかの判定が失敗した場合、スクリプトは非ゼロ終了します。
