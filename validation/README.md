# 四つの平均--分散解概念の検証

付録A.3に対応する検証コードです。van Staden, Dang and Forsyth (2021) Table 5.1にはPCMVだけでなく、DOMV、cTCMV、dTCMVの数値例も掲載されているため、四解概念すべてを外部再現の対象とします。そのうえで、本稿固有の制約付き月次実装を独立前進伝播と格子監査で検証します。

| モデル | 外部再現 | 本稿実装の内部監査 |
| --- | --- | --- |
| PCMV | reflected lognormal閉形式から公表16項目を再計算し、200万経路Monte Carloでも13項目を確認 | 後退・前進モーメント、質量保存、境界量 |
| DOMV | 正規終端分布と目標平均に対応する係数から公表16項目を再計算し、Monte Carloでも確認 | 後退・前進モーメント、質量保存、境界量 |
| cTCMV | 正規終端分布と目標平均に対応する係数から公表16項目を再計算し、Monte Carloでも確認 | 後退・前進モーメント、質量保存、境界量 |
| dTCMV | 公表平均・標準偏差から対数正規終端分布を同定し、アンカーを除く13項目を再計算してMonte Carloでも確認 | 基準格子のモーメント・質量監査と、`x_max=900` 対 `x_max=2100` の尾部安定性 |

## dTCMV外部再現の範囲

dTCMVの終端富が対数正規分布となることは同論文に示されていますが、時変係数を決める数値積分方程式の解経路はTable 5.1には掲載されていません。このため、外部再現では公表平均と標準偏差を分布同定のアンカーとして使い、中央値、歪度、超過尖度、VaR、CVaR、下回り確率、条件付き平均を独立に再計算します。これは公表終端分布の再構成であり、未公表の時変係数経路そのものを再現したという主張ではありません。制御経路は本稿ソルバーの内部監査で別途確認します。

## 実行

リポジトリのルートで実行します。

```bash
python validation/external_validation_vanstaden2021_all_mv.py
python validation/run_all_validations.py
```

`run_all_validations.py`は外部再現も再計算し、主な出力を `results/validation/` に保存します。現行判定ではPCMV 65/65、DOMV 65/65、cTCMV 65/65、dTCMV 67/67項目が通過します。いずれかの判定が失敗した場合、スクリプトは非ゼロ終了します。
